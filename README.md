# 🎓 Student Performance Prediction (Lasso Regression)

## 📌 Objective
- Predict student final scores
- Identify key influencing factors
- Automatically remove irrelevant features using Lasso Regression

## ⚙️ Tech Stack
- Python
- Scikit-learn
- Streamlit

## 📊 Features Used
- Hours_Studied
- Attendance
- Sleep_Hours
- Previous_Scores
- Internet_Usage
